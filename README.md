# OSERA

A FINOS community project.

## Website

The project includes a website in the `website/` directory, based on the [FINOS ai-website](https://github.com/finos/ai-website) skeleton. The homepage is plain HTML in [`website/index.html`](website/index.html). It is currently a one-page site and will be deployed at [osera.finos.org](https://osera.finos.org).

### Local development

```sh
cd website
npm install
npm run start
```

### Production build

```sh
cd website
npm run build
```

Built output is written to `website/build/`.
